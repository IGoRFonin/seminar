Welcome, jQuery
javascript – скриптовый язык, который поддерживают большинство современных браузеров
jQuery(jq) – специальная библиотека, написанная на javascript, основное назначение которой заключается в обеспечении правильного взаимодействия между JavaScript и HTML.

"$" или "jQuery" - даёт понять компилятору, что работа ведётся через библиотеку jquery
Взаимодействия в jQuery основаны на селекторах, функциях и событиях.

$(селектор).обработчик_события(function(){
код_обработчика_события
});
Примеры вобора селекторов


$("p").обработчик...
$(".news").обработчик...
$("#list").обработчик...
$("#list li").обработчик...
$("#list li > a").обработчик...
$("[name='PHONE_FIELD']").обработчик...
$("[data-id]").обработчик...
$("div:first-child").обработчик...
Пример события.

$(".button").on( 'click', function(){
$(this).css('color', 'orange');
});
Перебор элементов.

$(".news li").each(function(index, element){

}
});
